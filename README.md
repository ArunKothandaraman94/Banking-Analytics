# 🏦 Czech Banking Analytics Dashboard

> End-to-End Banking Analytics Project using **SQL, PostgreSQL, Python, and Power BI**

---

# 📌 Project Overview

This project analyzes customer, account, loan, transaction, and card data from a Czech bank to uncover business insights that support strategic decision-making.

The project demonstrates the complete Business Intelligence workflow:

- Data Cleaning
- SQL Data Analysis
- Python Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- Business Storytelling

---

# 🎯 Business Problem

Banks generate millions of transactions every year. Without analytics, it becomes difficult to answer important business questions such as:

- Which customer segments generate the highest activity?
- What is the loan portfolio status?
- Which regions have the highest customer concentration?
- How have transactions changed over time?
- What customer demographics should the bank target?

This project transforms raw banking data into meaningful business insights using SQL, Python, and Power BI.

---

# 📂 Dataset

The project uses the **Czech Financial Dataset (Berka Dataset)** containing:

# 💡 Business Insights

### 1. Customer Demographics
- The customer base is concentrated within a few major districts.
- Adults between the working-age groups represent the largest customer segment.

### 2. Loan Portfolio
- Most loans fall into good standing, indicating a healthy loan portfolio.
- A smaller proportion of loans require closer monitoring due to their status.

### 3. Transaction Trends
- Transaction volume changes over time, helping identify seasonal activity patterns.
- Monthly trend analysis provides insights into customer banking behavior.

### 4. Card Usage
- One card type dominates customer usage, indicating customer preference.
- Premium card adoption remains comparatively low.

### 5. Business Intelligence
- Combining customer, loan, and transaction data enables better decision-making for marketing, lending, and customer retention strategies.
